
<h1 align="center">
  The Rolling Stone
</h1>

<p>
Web-app is an  Online version of Rolling Stone magazine with the cover of NWA Straight outta Compton. The program feels like a magazine. The user can play any song they would like.  
</p>

## Run Program 

<pre>
npm run start
</pre>
